RFIND

Smart RFID-enabled inventory system integrating ESP32 microcontrollers, real-time tracking, web dashboards, and IoT automation.

DTI Best User-Centred Approach Nominee

RFIND was nominated for the DTI Best User-Centred Approach Award for its practical and intuitive inventory management experience.

System Demonstration

RFIND combines RFID-based inventory localization, embedded firmware, wireless communication, and real-time web monitoring into a single integrated inventory management platform.

The system was designed to improve usability, reduce manual inventory tracking, and provide a more intuitive real-time inventory experience through both physical and digital interfaces.

Overview

RFIND is an IoT-based smart inventory shelf that automates inventory management using RFID technology and ESP32 microcontrollers.

Each item is tagged with an RFID label, allowing the system to detect when an item is borrowed or returned while displaying its location through LED indicators.

All inventory activity is logged automatically to Google Sheets while the dashboard updates in real time through the Google Sheets API.

Web Dashboard + Real-Time Tracking

A responsive web dashboard was developed using React, TypeScript, and Vite to visualize inventory activity and item locations in real time.

The dashboard supports live synchronization, search and filtering systems, inventory monitoring, and LED control communication with ESP32 devices.

Embedded Systems + Hardware Integration

RFIND integrates RFID readers, ESP32 microcontrollers, LEDs, and wireless communication systems into a full hardware-software prototype.

Embedded firmware was implemented in Arduino C++ to manage RFID scanning, device communication, inventory detection, and LED guidance systems.
